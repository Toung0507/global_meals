# Global Meals — 系統配色規範

> 本文件記錄已確定使用的配色核心，
> 未來所有新頁面請依照使用者的**身份別**選用對應的色彩方案，確保視覺一致性。

---

## 身份別對應色彩方案總覽

| 身份別 | 色彩方案名稱 | 主視覺感受 |
|--------|------------|----------|
| 管理系統（老闆、分店長、員工） | 深藍金 — 沉穩專業 | 深海軍藍底 × 金黃點綴 |
| 客戶（會員、訪客） | 暖橘奶油 — 親切美食 | 活力橘紅底 × 奶油白底色 |

## 方案一：深藍金（管理系統）

**適用頁面：** 管理系統登入、老闆後台、分店長後台、員工操作頁面

**已使用於：** `staff-login.component.scss`

### 色票

| 角色 | 色彩名稱 | HEX 色碼 | 用途說明 |
|------|---------|---------|---------|
| 主色 | 深海軍藍 | `#1E3A5F` | 左側品牌欄背景、按鈕主色、active 狀態 |
| 輔助色 | 中藍 | `#2C5282` | 按鈕漸層起始色、hover 狀態 |
| 點綴色 | 亮金黃 | `#F0A500` | 品牌英文名稱、數值資訊、重點強調 |
| 主要文字 | 深藍黑 | `#1A2840` | 標題、輸入框文字 |
| 次要文字 | 藍灰 | `#4A5A70` | 說明文字、副標題、頁尾備註 |
| 頁面背景 | 淡藍灰 | `#F4F6FA` | 整頁底色、輸入框底色 |
| 卡片背景 | 白色 | `#FFFFFF` | 右側表單欄、卡片底色 |
| 邊框線條 | 藍灰邊框 | `#D4DCE8` | 輸入框邊框、分隔線、tab 邊框 |

### 延伸色（特定情境使用）

| 用途 | HEX 色碼 | 說明 |
|------|---------|------|
| 未選中標籤背景 | `#D4DCE8` | 角色切換 tab 未選中狀態 |
| 未選中標籤文字 | `#6A7A90` | 角色切換 tab 未選中文字 |
| hover 加深邊框 | `#BEC8D8` | 導覽列 hover 背景 |
| placeholder 文字 | `#A0AABB` | 輸入框佔位提示文字 |
| 白色透明疊加 | `rgba(255,255,255,0.10)` | 深色背景上的半透明框線、資訊列 |

### 使用規則

- 左側品牌視覺區塊 → 主色 `#1E3A5F` 作為背景
- 主要行動按鈕 → `#2C5282` 到 `#1A3558` 漸層
- 強調數值、品牌標語 → 點綴色 `#F0A500`
- 輸入框預設邊框 → `#D4DCE8`，focus 時改為主色 `#1E3A5F`

| 用途 | HEX 色碼 | 說明 |
|------|---------|------|
| placeholder 文字 | `#A89080` | 輸入框佔位提示文字 |

| 密碼遮罩點 | `#A84210` | 密碼欄的 •••• 顯示 |

| 成功提示背景 | `rgba(46,125,50,0.10)` | 操作成功 toast 背景 |

- 左側品牌視覺區塊 → 暖色漸層背景（橘紅系）

## 客戶端 Loading 過場動畫（v4 分割卡片版本）

**元件路徑：** `src/app/shared/customer-loading/`

**觸發時機：**
- `staff-login` → 點擊「客戶入口」→ `loadingService.showCustomerLoading()` → 導向 `/customer-login`
- `customer-login` → 登入成功 → `loadingService.showCustomerLoading()` → 導向 `/customer-member`

**版面結構：**

```
全螢幕奶油白遮罩（#FFFAF3, 0.97 不透明）
└── 分割卡片（flex row，最大 840px，圓角 24px）
    ├── 左側面板（橘色漸層：#D95C1A → #A84210）
    │   ├── Logo（圓形，96px，白色邊框）
    │   ├── 懶 飽 飽（白色 #FFFFFF，LXGW WenKai TC，36px，字距 8px）
    │   ├── Global Meals（金黃 #F0A500，Fraunces serif，18px，字距 4px）
    │   └── ✦ 懶懶吃，飽飽樂 ✦（暖奶油 #FFF5E6，LXGW，13px，半透白圓角徽章）
    └── 右側面板（白色 #FFFFFF）
        ├── Lottie 動畫容器（180×180px，lottie-web npm）
        ├── 進度條（橘紅→金黃漸層，1.4 秒填滿至 95%）
        ├── 三個跳動點點（橘紅 #D95C1A，nth-child 延遲 0/0.2/0.4s）
        └── 狀態文字（中棕 #6A4830，Noto Sans TC，13px）
```

**三層品牌文字配色（左側橘色面板上）：**

| 文字元素 | 色彩 | 說明 |
|---------|------|------|
| `懶 飽 飽`（中文品牌名） | `#FFFFFF` 純白 | 最高對比，主視覺 |
| `Global Meals`（英文副標） | `#F0A500` 亮金黃 | 品牌點綴色，溫暖貴氣 |
| `✦ 懶懶吃，飽飽樂 ✦`（標語徽章） | `#FFF5E6` 暖奶油 | 比純白更柔和，三色形成層次 |

**動畫技術：**

| 效果 | 技術 | 說明 |
|------|------|------|
| 入場動畫 | GSAP | 遮罩淡入 0.4s + 卡片彈出 0.7s（elastic.out） |
| 食物動畫 | lottie-web npm | `lottie.loadAnimation()`，SVG 渲染，loop + autoplay |
| 進度條 | CSS @keyframes | `progressFill`，1.4s ease-out，填至 95% |
| 跳動點點 | CSS @keyframes | `dotBounce`，1.2s infinite，nth-child 錯開延遲 |

**Logo 來源：** `public/assets/logo圓形.png` → Angular 打包後路徑為 `/assets/logo圓形.png`
