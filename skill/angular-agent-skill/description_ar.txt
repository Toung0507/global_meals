تطوير Expert Angular وTypeScript وTailwind باستخدام أنماط حديثة قائمة على الإشارات ومكونات مستقلة وتدفق تحكم مدمج. يُستخدم عند العمل على مشاريع Angular، أو إنشاء المكونات، أو الخدمات، أو التوجيهات، أو الأنابيب، أو كتابة القوالب، أو إدارة الحالة بالإشارات، أو التعامل مع النماذج، أو التوجيه، أو طلبات HTTP، أو الاختبار، أو إمكانية الوصول (a11y)، أو ARIA، أو أي تعليمات برمجية متعلقة بـ Angular. استخدمه أيضًا عند مراجعة التعليمات البرمجية Angular، أو تصحيح مشكلات Angular، أو اتخاذ قرارات معمارية، أو تنفيذ أنماط واجهة المستخدم التي يمكن الوصول إليها مثل مصائد التركيز، أو إعلانات قارئ الشاشة، أو التنقل عبر لوحة المفاتيح، أو سمات ARIA في تطبيقات Angular. استخدم هذه المهارة عند إنشاء مكونات يمكن الوصول إليها بدون رأس باستخدام @angular/aria - بما في ذلك علامات التبويب، أو الأكورديون، أو القوائم، أو أشرطة القوائم، أو أشرطة الأدوات، أو مربعات القائمة، أو القوائم المنسدلة المحددة، أو التحديد المتعدد، أو الإكمال التلقائي، أو مربع التحرير والسرد، أو الشبكات، أو طرق عرض الشجرة - حتى إذا لم يذكر المستخدم صراحة "الأغنية" أو "إمكانية الوصول".
