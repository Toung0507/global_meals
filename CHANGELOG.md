# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0.0] - 2026-04-09

### Added
- **POS 商品卡重設計**：移除 SVG 圖示區塊與所有 badge 標籤（熱銷 / 新品 / 低庫存），改為純文字精美餐飲按鈕，只顯示品名、NT$ 價格與 `+` 符號
- **POS 現金計算機 overlay**：選擇現金付款後跳出全螢幕數字鍵盤，支援快速金額按鈕（向上取整至 100/500/1000）、即時找零計算，找零不足時確認按鈕 disabled
- **POS 會員折扣券累積面板**：查詢到會員後，顯示仿客戶端側邊欄的折扣券紀錄區塊，含進度條與「可使用 / 累積中」badge
- **滿額贈品下拉（POS & 客戶端）**：移至購物車小計區（小計 ≥ $300 顯示），新增「不需要滿額免費贈品」為第一選項；已選贈品於訂單品項顯示為 $0 列（無增減按鈕）
- **活動優惠兩層下拉（POS & 客戶端）**：小計達活動門檻後出現；第一層選擇活動，第二層從該活動專屬贈品清單挑選（三個活動各有不同選項）
- **訂單預覽 Modal（客戶端）**：「進入結帳頁面」改為先彈出確認視窗，客戶可最後確認品項，按「上一步」返回購物車修改，或「確認建立訂單」前往付款頁

### Changed
- **移除 POS 營業稅**：台灣無附加營業稅，移除 5% 稅金欄位，合計 = 小計
- **POS 會員查詢手機號碼比對**：比對前去除所有 `-`，`0912345678` 可成功比對到存為 `0912-345-678` 的會員
- **折扣券邏輯修正（POS & 客戶端）**：不使用折扣券時次數維持在 10 不重置；使用折扣券後從第 1 次重新累積；`Math.min(count + 1, 10)` 避免超過 10
- **付款頁返回按鈕（客戶端）**：「返回」改為「取消本次訂單，返回首頁」，清空購物車並導回首頁 tab
- **`tsconfig.json`**：加入 `"rootDir": "./src"` 修正 TypeScript 輸出路徑警告

## [0.1.0.0] - 2026-04-05

### Added
- **客戶端主頁（Customer Home）**：完整下單流程、菜單瀏覽、食物照片展示、今日優惠區塊、Hero 區塊
- **客戶會員中心（Customer Member）**：會員資料頁、頭像上傳、訂單歷史
- **客戶登入/註冊（Customer Login/Register）**：表單驗證、sessionStorage 持久化
- **客戶訪客模式（Customer Guest）**：無帳號瀏覽體驗
- **員工 POS 終端（POS Terminal）**：品牌改版、商品分類搜尋、點餐結帳流程
- **管理後台（Manager Dashboard）**：訂單看板、營業統計、員工管理
- **員工登入（Staff Login）**：角色驗證（老闆/員工）
- **讀取動畫（Loading）**：客戶端分割卡片動畫（v4）、員工端 Loading 元件
- **設計系統基礎**：`_design-tokens.scss` 設計代字、Bootstrap 5.3.8 整合
- **Lottie 動畫**：本機 scan-to-order 動畫資源
- **訂單服務（order.service.ts）**：前端訂單狀態管理
- **Auth 服務**：sessionStorage 持久化、updateProfile() 同步更新
- **路由設定**：三種身份角色路由（客戶/員工/老闆）

### Changed
- `app.component.html`：主路由框架重構，整合角色導向導覽
- `README.md`：更新專案說明與開發狀態
- `LAPTOP_SETUP.md`：新增環境設定文件
