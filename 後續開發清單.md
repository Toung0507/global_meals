## 🟡 中優先（Demo 後第一週）

### 2. 菜單圖片串接

**位置：** `customer-home.component.html` 的各 `.menu-item-img` class

**現況：** 圖片使用 CSS 背景圖，非動態載入。

**方案：**
```ts
// 每張商品卡加入 [style.background-image] 綁定
getProductImageUrl(id: number): string {
  return `${API_CONFIG.BASE_URL}/lazybaobao/products/${id}/image`;
}
```
後端 `GET lazybaobao/products/{id}/image` 回傳 Base64，前端用 `data:image/jpeg;base64,` prefix 顯示。

### 3. BranchService 加入 globalAreaId

**位置：** `src/app/shared/branch.service.ts`

**問題：** `BranchService` 目前只管 country（TW/JP/KR），沒有 `globalAreaId` 欄位，
導致菜單 API 只能 hardcode `globalAreaId = 1`。

**做法：** 在 `CountryConfig` 或 `BranchService` 加入 `globalAreaId: number`，
並在 QR Entry 時由 URL 參數帶入（`?area=1`）。
